# GS-8

## Score Type Decision

- score_type: binary
- rationale: 约束遵循评测只关心"是否满足"，1=满足，0=不满足，半对算错。

## Judge Instruction

judge 需要判断回答是否适合约束指定的目标受众阅读。关注：用词和概念难度是否与受众专业水平匹配、解释详略是否恰当、语气和呈现方式是否符合该受众的阅读场景。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 回答的用词、深度、语气与目标受众匹配 |
| 0 | 与目标受众不匹配：对专业受众过于浅显，或对非专业受众过于晦涩，或语气与场景不符 |

## Checklist

- 回答的专业深度是否与目标受众匹配？
- 如果面向非专业受众，专业术语是否有解释？
- 语气是否符合该受众的阅读场景？

## Edge Cases

1. **面向"管理层"**：应简明重结论。过于学术化或过于口语化均判 0。
2. **"高净值客户"**：不具备专业背景但认知能力较高，过度简化或堆砌术语均判 0。
