# FS-21

## Score Type Decision

- score_type: binary
- rationale: 1=满足，0=不满足，半对算错。

## Judge Instruction

judge 判断回答是否包含竞品或同业对比分析。必须有明确的对比对象和对比维度，不能只提到竞品名字而无对比。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 回答包含至少一组明确的竞品/同业对比，有对比维度和结论 |
| 0 | 未包含对比分析，或仅提及竞品名称但无实质对比 |

## Checklist

- 是否明确指出了对比对象（竞品/同业）？
- 是否有具体的对比维度（如市场份额、产品特性、财务指标等）？
- 是否有对比结论或差异分析？

## Edge Cases

1. **隐含对比**：未用"对比"一词但用了"相比""优于""不及"等表达，实质上构成对比，判 1。
2. **只列名字**："主要竞品有 A、B、C"但无任何维度对比，判 0。
