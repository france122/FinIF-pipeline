# FS-18

## Score Type Decision

- score_type: binary
- rationale: 1=满足，0=不满足，半对算错。

## Judge Instruction

judge 判断回答是否重点包含/分析了约束指定的内容方向。"重点"意味着有实质性分析篇幅，不是一句话带过。如果约束指定了多个方向，每个都需有实质讨论。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 对指定的每个内容方向都有实质性分析 |
| 0 | 完全未涉及、仅一句话带过、或任一指定方向缺失 |

## Checklist

- 约束指定的每个分析方向是否都有实质讨论？
- 是否有具体分析内容而非仅标题/提及？

## Edge Cases

1. **多方向全覆盖**：指定 3 个方向只覆盖 2 个，判 0。
2. **隐含分析**：没分章节但论述中实质覆盖了所有方向，判 1。
