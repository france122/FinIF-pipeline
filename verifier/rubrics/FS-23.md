# FS-23

## Score Type Decision

- score_type: binary
- rationale: 1=满足，0=不满足，半对算错。

## Judge Instruction

judge 判断回答是否包含预算或成本分析。必须有具体的金额/费用/成本数据或分项说明，不能只说"需要考虑成本"。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 回答包含具体的预算分配、成本构成或费用分析 |
| 0 | 未涉及预算/成本，或仅泛泛提及"需控制成本"而无具体内容 |

## Checklist

- 是否有具体的金额、费用项或预算分配？
- 是否有成本分项说明或预算表？

## Edge Cases

1. **概算级别**：给出了大致预算范围（如"总预算约 50 万元"）但有分项，判 1。
2. **只说原则**："应控制成本在合理范围内"无任何数字，判 0。
