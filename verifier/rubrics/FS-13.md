# FS-13

## Score Type Decision

- score_type: binary
- rationale: 1=满足，0=不满足，半对算错。

## Judge Instruction

judge 判断回答是否避免使用金融术语的英文缩写（如 ROE、PE、EBITDA、EPS、GDP、CPI、IPO、M&A、ABS、ETF 等）。所有此类术语应使用中文全称表达。这是一个反直觉约束——金融文本中通常大量使用英文缩写，此约束要求全部替换为中文。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 全文未出现任何金融术语英文缩写，所有术语均使用中文全称 |
| 0 | 出现了至少一个金融术语英文缩写 |

## Checklist

- 全文是否出现了 ROE/ROA/PE/PB/EPS/EBITDA/GDP/CPI/PPI/IPO/M&A/ABS/ETF/REITs/NPL/LPR 等常见金融缩写？
- 是否有不太常见但仍属于金融领域的英文缩写（如 WACC、DCF、CAPM、NAV、YTD、QoQ、YoY、CAGR 等）？
- 通用英文词（如 AI、IT、App）不属于"金融术语缩写"，不应扣分。

## Edge Cases

1. **通用技术/非金融缩写**：AI、IT、App、5G、WiFi 等不属于金融术语缩写，出现不扣分，判 1。
2. **中英混用且给了全称**：如"净资产收益率（ROE）"，括号内出现了缩写，仍判 0——约束要求不得使用缩写。
3. **公司名/产品名中的英文**：如"ETF基金"作为产品名而非术语缩写，仍判 0——ETF 本身是金融缩写。
4. **全文无任何金融术语**：回答内容不涉及任何需要缩写的术语，判 1。
