# GS-9

## Score Type Decision

- score_type: binary
- rationale: 1=满足，0=不满足，半对算错。

## Judge Instruction

judge 判断回答的语气是否达到约束指定的效果（诚恳/有感染力/有说服力）。关注措辞是否传达目标语气、是否机械模板化、论据是否充分。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 语气与约束要求一致 |
| 0 | 语气不符：平淡机械、套话堆砌、或与目标语气相反 |

## Checklist

- 整体语气是否与约束要求一致？
- 是否有模板化套话降低真实感？
- 如果要求说服力，是否有具体论据支撑？

## Edge Cases

1. **金融"说服力"**：夸大营销但缺乏实质论据，判 0。
2. **模板化诚恳**："我们真诚地建议"但内容空洞，判 0。
