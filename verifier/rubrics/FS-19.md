# FS-19

## Score Type Decision

- score_type: binary
- rationale: 1=满足，0=不满足，半对算错。

## Judge Instruction

judge 判断回答是否包含约束指定的多类型内容板块（如分析、评估、建议、对比等）。每个指定板块都需出现。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 回答包含了约束指定的所有内容板块类型 |
| 0 | 缺少任一指定板块，或板块存在但无实质内容 |

## Checklist

- 约束指定的每种板块类型（分析/评估/建议/对比等）是否都出现？
- 每种板块是否有实质内容而非空壳标题？

## Edge Cases

1. **板块融合**：如果"分析"和"评估"融合在同一段落中但内容都覆盖了，判 1。
2. **只有标题无内容**：列了板块标题但下面无实质内容，判 0。
