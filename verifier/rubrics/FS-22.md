# FS-22

## Score Type Decision

- score_type: binary
- rationale: 1=满足，0=不满足，半对算错。

## Judge Instruction

judge 判断回答是否包含趋势预测或未来展望。必须有对未来方向/发展/变化的明确判断，不能只描述现状。

## Scoring Rubric

| 分值 | 描述 |
|------|------|
| 1 | 回答包含对未来趋势、发展方向或前景的明确判断或预测 |
| 0 | 仅描述现状或历史，未涉及任何前瞻性内容 |

## Checklist

- 是否有对未来的明确判断（如"预计""将会""趋势是"）？
- 预测是否有逻辑支撑而非凭空臆断？

## Edge Cases

1. **模糊展望**："未来前景广阔"一句话带过无具体内容，判 0。
2. **条件预测**："如果 X 发生则 Y 可能出现"——有条件的前瞻性分析，判 1。
