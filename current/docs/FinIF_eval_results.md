## 分析

### 1. 总体排名与模型代际

排名完全符合模型代际预期：**GPT-5.4/5.2 >> GPT-5/DS-V4 >> GPT-5.1 >> Qwen3-8B**。

- GPT-5.4/5.2 在 Prompt-level（87-88%）和 Instruction-level（95.7-96.0%）上遥遥领先，Hard 和 Soft 通过率均在 94%+ 以上
- DS-V4 四个变体紧密聚集在 73-77% / 89.7-90.7% 区间，差异不大
- GPT-5.1（69%）表现反常地低于 GPT-5（77%），主要短板在 N1 字数控制（33%）和 4 条约束 case（35%）
- Qwen3-8B（62% / 84.3%）垫底，SFT 目标提升空间约 13-25pp
- **Qwen3-8B-SFT（77% / 91.3%）通过 SFT 跃升至第 3 梯队**，追平 GPT-5 和 DS-V4-Pro-Think

### 2. 约束数量与难度梯度

所有模型呈现清晰的**约束数-通过率负相关**：

- 1 条约束：大部分模型 93-100%（设计目标 ≥ 90%）
- 5 条约束：GPT-5.4/5.2 仍有 80%，DS-V4 降至 47-60%，Qwen3-8B 仅 33%

GPT-5.4/5.2 在 5 条约束下仍保持 80% prompt-level，说明其指令遵循能力明显优于 DS-V4 系列。

### 3. Hard vs Soft 差异

- GPT-5.4/5.2 的 Soft 通过率（97-98%）显著高于 Hard（94.7%），说明 LLM judge 对这些模型的主观评价更宽松
- Qwen3-8B 是唯一 Hard > Soft 的模型（86.1% vs 81.4%），说明小模型在风格/连贯性等软约束上的差距比格式约束更大

### 4. 弱项标签

按 9 模型平均通过率，最难的三个标签：


| 标签      | 平均    | 最低模型                 | 说明        |
| ------- | ----- | -------------------- | --------- |
| N3 精度   | 61.6% | Qwen3-8B (18%)       | 小数位数控制    |
| F4 JSON | 66.7% | DS-Flash (33%)       | JSON 格式输出 |
| S3 连贯性  | 67.9% | DS-Flash/Qwen3 (33%) | 段落逻辑连贯    |


这三类也是 SFT 训练数据中加权采样的目标（TAG_BOOST = 3.0）。

### 5. Thinking 模式影响


| 对比                   | Prompt-level Δ | Instruction-level Δ |
| -------------------- | -------------- | ------------------- |
| Flash-Think vs Flash | -2pp           | -0.3pp              |
| Pro-Think vs Pro     | +1pp           | +0.7pp              |


DeepSeek 的 thinking 模式对 IF 能力无显著提升，甚至 Flash-Think 反而略低于 Flash——thinking 可能产生更长的推理过程但不一定更好地遵循格式约束。

### 6. GPT 系列演进

GPT-5 → 5.1 → 5.2 → 5.4 的 IF 能力非单调：


| 版本      | Prompt-level | 特点                              |
| ------- | ------------ | ------------------------------- |
| GPT-5   | 77%          | reasoning: medium，token 开销约 2×  |
| GPT-5.1 | 69%          | N1 字数控制退化（33%），4 条约束 case 仅 35% |
| GPT-5.2 | 87%          | 大幅提升，接近 5.4                     |
| GPT-5.4 | 88%          | 系列最佳，Soft 98.2%                 |


GPT-5.1 的退化和 5.2 的跳升说明 IF 能力与模型通用智能不完全正相关，验证了 FinIF 衡量的确实是**指令遵循**这一独立维度。

### 7. Qwen3-8B SFT 效果

SFT 后 Qwen3-8B 在所有维度均有显著提升：


| 指标                | Baseline | SFT   | Δ       |
| ----------------- | -------- | ----- | ------- |
| Prompt-level      | 62%      | 77%   | +15pp   |
| Instruction-level | 84.3%    | 91.3% | +7.0pp  |
| Hard              | 86.1%    | 92.0% | +5.9pp  |
| Soft              | 81.4%    | 90.3% | +8.8pp  |
| T1                | 87.5%    | 93.2% | +5.7pp  |
| T2                | 82.4%    | 94.1% | +11.7pp |
| T3                | 80.4%    | 84.5% | +4.1pp  |


**关键发现**：

- **Prompt-level +15pp** 是最显著的提升维度，从 62% 跃至 77%，追平 GPT-5 和 DS-V4-Pro-Think
- **Soft 提升（+8.8pp）大于 Hard（+5.9pp）**，说明 SFT 有效改善了风格/连贯性等主观维度——Baseline 时 Qwen3-8B 是唯一 Hard > Soft 的模型，SFT 后 Hard/Soft 基本持平
- **T2 提升最大（+11.7pp）**，达到 94.1%，甚至超过 GPT-5（89.8%）和所有 DS-V4 变体，说明综合分析类任务受益于 SFT 训练数据中的多约束梯度采样
- **T3 仍是短板（84.5%）**，与 DS-V4 系列（85.7-87.5%）尚有差距，复杂推理和跨表核验需要更深层的能力
- **弱项标签改善明显**：N3 精度 18%→82%（+64pp）、F4 JSON 50%→100%（+50pp）、S3 连贯性 33%→78%（+45pp），验证了 TAG_BOOST 加权采样策略的有效性
- **N1 字数控制退化（83%→67%）**，可能因 SFT 训练数据中字数约束的多样性不足
- **5 条约束 case 从 33% 提升至 67%（+34pp）**，但仍显著低于 GPT-5.4（80%），多约束并行遵循仍有提升空间
